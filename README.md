# ActoDo - Enhanced To-Do List Application

ActoDo is a modern and professional task management application built using React.js and Tailwind CSS. The application helps users organize daily activities efficiently with advanced task tracking, priority management, analytics, and a responsive user experience.

## Features

### Task Management

* Create, edit, and delete tasks
* Mark tasks as completed
* Assign High, Medium, or Low priority levels
* Set due dates for tasks
* Visual status indicators for task completion

### Filtering and Sorting

* Real-time task search
* Filter tasks by priority
* Sort tasks by newest first or priority level
* Show or hide completed tasks

### Analytics Dashboard

* Progress bar with completion percentage
* Total tasks count
* Completed tasks count
* High-priority tasks count

### Data Persistence

* Local storage integration
* Automatic saving of tasks and user data
* Persistent state across browser sessions

### User Interface

* Professional Navy Blue and Sky Blue theme
* Smooth animations and transitions
* Dark mode support
* Fully responsive design
* Accessibility-friendly components

### Form Validation

* Input validation
* Password confirmation validation
* Error handling and user feedback
* Demo credential support

## Technologies Used

* React.js
* Tailwind CSS
* JavaScript (ES6+)
* Local Storage API
* React Hooks

## Future Enhancements

* User authentication with backend integration
